.. _section-upgrade:

=======
Upgrade
=======

Before an upgrade of this extension, please read this section carefully.


.. _section-versioning:

Versioning
==========

This extension follows the rules of `Semantic Versioning
<https://semver.org/>`_.


.. _section-preparation:

Preparation
===========

Before an upgrade, read the changelog of all versions between the currently
installed and the target version (that one you want to upgrade to). They can be
found :ref:`here <section-changelog>`.
